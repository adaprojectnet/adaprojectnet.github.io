ABOUT


Every Question. Every Answer.

The ADA Project came about when thinking of how textbooks can only display an answer.

There are forums for discussing questions and different websites which can show solutions.

The ADA Project aims to combine all of this in an organised and collaborative manner with the aim of listing Every Question and Every Answer

What do we mean by Every Question?
We aim to list all Textbook style questions as well as past papers and user submitted questions.

What do we mean by Every Answer?
Each question has a simple answer, a worked through solution, a video link, for a more in depth answer, and a chance for discussion, if there are any follow on questions.

The aim is to have the best possible solutions too, and these can be constantly updated.

This is not possible by one person or even a group of people. It aims to be ever expanding and the more contributions, the better the result.

Finally, the ADA Project aims to be free forever - education should be accessible and free to anyone.

We hope you enjoy the ADA Project.
